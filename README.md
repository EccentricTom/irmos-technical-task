# Irmos Technical task

This is the README detailing how to setup, run and test the takehome task.

## Setup Instructions

You can set up this repository using either [uv](https://github.com/astral-sh/uv) or standard `pip`.

### Using uv

```bash
uv sync
```

### Using pip

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
pip install -r requirements.txt
```
